## 概要

- SPR(Sapphire Rapids)で対応予定。
- 命令の最後にph()やsh()がつく。
- FP16のdenomal(非正規化数)はフルスピードで処理される。
- MXCSRの設定に影響されない。

### evex

0x62の次の`P0 = [R:X:B:R':0:0:m:m]`が`P0 = [R:X:B:R':0:m:m:m]`に拡張された。

### displacement scaling

- full
  - N = 2 ; FP16 broadcast
  - N = 4 ; 複素FP16 broadcast
  - N = 16, 32, 64 ; full vector
- fullmem
  - N = 16, 32, 64 ; full vector
- scalar
  - N = 2 ; FP16
  - N = 4 ; 複素FP16
- half
  - N = 2 ; broadcast
  - N = 8, 16, 32 ; half vector
- halfmem
  - N = 8, 16, 32 ; half vector
- quarter
  - N = 2 ; broadcast
  - N = 4, 8, 16 ; 1/4 vector
